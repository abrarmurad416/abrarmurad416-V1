Currency Exchange Calculator

Language used: C++

Straightforward and efficient calculator that converts 10 different currency values to CAD and vice versa. Additionally, the calculator aggregates the total price of each currency and displays it to the user in to CAD and out from CAD.



Online Vendor System

Language used: C++

Premium-themed online vendor system that sells a collection of exotic apples. Each apple has a short description that describes its location, taste/feel, availabilty, and supplementary info. Futhermore, the code displays a grid-based selection system that incorporates the apple prices, quantity for selection, and international or national fees for shipping. A digital receipt is outputted once the user submits the order.
